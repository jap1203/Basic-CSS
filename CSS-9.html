HTML CSS Example

Cascading style sheets, fondly referred to as css, is a simple design language intended to simplify the process of making web pades presentable.

CSS handles the look and feel part of a web pages, Using CSS, you can control the color the text, the style of fonts, the spacing between paragraphs, how colums are sized and laid out, what background images or colors are used, as well as a variety of other effects.

CSS is easy to learn and understand but it provides powerful control over the precentation of an HTML document. Most commonly, CSS is combined with the markup language HTML or XHTML.
